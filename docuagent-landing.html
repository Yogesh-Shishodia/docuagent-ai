How It Works

Up and running in under 10 minutes

1

Install Ollama and pull the model

Ollama runs a local LLM server on your Mac, Linux, or Windows machine. The model (llama3:8b, ~4.7GB) downloads once and stays on your hardware forever.

$ brew install ollama && ollama pull llama3:8b
2

Clone the repo and install dependencies

Pure Python — no cloud keys, no vendor registration. ChromaDB stores your vectors locally on disk, sentence-transformers runs the embedding model on-device.

$ git clone github.com/YOU/docuagent-ai && pip install -r requirements.txt
3

Launch and start uploading documents

Drag in PDFs, Word docs, spreadsheets, presentations, EPUBs, or paste any URL. Everything is chunked, embedded, and indexed locally in seconds.

$ streamlit run app.py
4

Ask questions, get cited answers

Every answer cites the exact source, page, and passage it was drawn from. No hallucination risk on source attribution. Click any citation to verify instantly.

Deployment

Three ways to get DocuAgent running

From a solo Mac install to a managed AWS instance for teams without on-prem infra — pick what fits your situation.

💻

Local / On-Prem

For: Legal, Healthcare, Defense, Consulting

The purest deployment — runs entirely on your laptop, workstation, or existing server. Nothing leaves your network.

  • Mac (Apple Silicon or Intel), Linux, or Windows
  • MacBook M2/M3 with 16GB RAM runs llama3:8b at 30–40 tok/s
  • One-command installer, no cloud dependency
  • Full air-gap capability for classified environments
  • Persistent vector store survives reboots
Best for: individuals and firms with their own hardware
☁️

AWS Private Cloud

For: Teams without on-prem GPU infra

Deploy DocuAgent on your own AWS account inside a private VPC. You own the instance, the data never leaves your tenancy.

  • Single EC2 instance (g4dn.xlarge ~$0.50/hr, or reserved)
  • No shared infra — your VPC, your S3, your keys
  • HTTPS access via Nginx + SSL from any browser
  • Auto-start on reboot via systemd
  • SOC2 / HIPAA BAA eligible (AWS covers infra layer)
  • Stop instance when not in use — pay only when running
Estimated: $50–150/mo for a team (g4dn.xlarge on-demand)
🐳

Docker / Self-Hosted

For: DevOps-comfortable teams, internal tooling

Single docker-compose up deploys the full stack on any Linux server or VM.

  • Compose file includes Ollama + DocuAgent + Nginx
  • Deployable on Hetzner, DigitalOcean, Azure, GCP, bare metal
  • Volume mounts preserve data across container restarts
  • Traefik or Nginx for HTTPS termination
  • Behind corporate VPN or Tailscale for team access
Infra: $20–80/mo depending on provider and GPU tier
